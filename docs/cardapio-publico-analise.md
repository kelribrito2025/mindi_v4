# Análise do Site de Referência - Cardápio Público

## Site Analisado
https://mandinhoburgernh.saipos.com/home

## Estrutura Visual Identificada

### Header
- Logo/nome do restaurante
- Navegação: Início, Pedidos, Perfil
- Botão de login

### Área Principal
- Banner/imagem de capa do restaurante
- Informações do estabelecimento
- Botão "Mais informações"
- Calculadora de taxa de entrega

### Categorias
- Lista horizontal de categorias (navegação rápida)
- Scroll horizontal para navegar entre categorias

### Cards de Produtos
- Imagem do produto à esquerda
- Nome do produto
- Descrição curta
- Preço
- Botão de adicionar ao carrinho

### Sidebar (Carrinho)
- Sacola de compras
- Lista de itens
- Subtotal, taxa de entrega, total
- Campo de cupom
- Botão finalizar pedido

## Plano de Implementação para Mindi

### Adaptações para Identidade Visual Mindi
- Cores: Laranja (#F97316) como cor primária
- Fontes: Inter (já usada no projeto)
- Bordas arredondadas consistentes
- Sombras suaves

### Estrutura da Página Pública
1. Header com logo Mindi e nome do restaurante
2. Banner/capa do restaurante
3. Informações básicas (horário, endereço)
4. Navegação de categorias (horizontal scroll)
5. Lista de produtos por categoria
6. Cada produto: imagem, nome, descrição, preço

### Sem Carrinho (Fase 1)
- Apenas visualização do cardápio
- Sem funcionalidade de pedido online
- Foco em exibir categorias e produtos
